操作提醒

1. 审批通过后，教师可直接下载公示 PDF，不再跳转第三方网盘。

2. 批量创建账号已迁移到 Supabase Edge Function，前端不再要求输入 Service Role Key。

3. 审批记录现在支持审批人和驳回原因，便于追溯。

4. 批量 PDF 导出支持按年级、学科、关键词、时间区间组合筛选，还可在 ZIP 中按年级 / 学科 / 教师自动分目录。
