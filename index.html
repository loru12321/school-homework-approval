操作提醒

1. 审批通过后，教师可直接下载公示 PDF，不再跳转第三方网盘。

2. 批量创建账号已迁移到 Supabase Edge Function，前端不再要求输入 Service Role Key。

3. 审批记录现在支持审批人和驳回原因，便于追溯。

4. 审批列表本身已支持按年级、学科、教师和时间区间筛选，便于先筛后批量处理。

5. 批量 PDF 导出支持按年级、学科、关键词、时间区间组合筛选，还可在 ZIP 中按年级 / 学科 / 教师自动分目录。

6. 批量 PDF 导出已迁到 Supabase 后台任务，生成完成后支持重复下载与失败重试。

7. PDF 模板可直接配置学校名称、页眉、落款和命名规则，后续换学校或换模板无需再改代码。

8. 导入历史作业时，系统会自动补全缺省字段并生成可追踪的文件编号。
